# 교사 연수 덱 — blueprint
- 폴더: `teacher-training/`   · 대상: 화학·과학 교사   · 형태: 인터랙티브 HTML 덱(deck-shell 기반)
- 성격: 개념 소개 + 라이브 시연 (설치는 간단히)   · 상태: building   · 최종수정: 2026-06-17

## 학습 목표 (연수 후 교사가)
1. 이 시스템이 "거시↔미시 루프로 수업을 한 장씩 만든다"는 핵심을 설명할 수 있다.
2. blueprint → slide-compose → flow-check 흐름과 각 단계의 역할을 안다.
3. 코드 없이 인터랙티브 시뮬·발문·형성평가가 만들어짐을 직접 보고, 시작 방법을 안다.

## 대상 오개념 (연수 참석자)
- "AI가 한 번에 다 만들어 준다" → 실은 한 장씩, 교사가 흐름을 통제.
- "코딩을 배워야 한다" → 말로 요청하면 검증된 조각으로 조립.
- "결과물이 조잡할 것" → 실측 데이터·교과서 표기·투사 가독성까지 규율로 보장.

## 서사 아크
공감(문제) → 해결(시스템 한 장 요약) → 원리(거시↔미시 루프 3단계) → 시연(시뮬·자동화) → 신뢰(교육 원칙·결과물) → 행동(시작법) → 점검(퀴즈) → 정리.

## 시각·언어 규칙
- 전부 한국어. design-system.md의 기존 클래스/토큰만 사용(새 CSS 금지).
- 흡열=빨강·발열=파랑 색 언어 유지. 칩 흐름(`.flowrow`)·카드(`.card`)·정리목록(`.sumlist`) 활용.
- 각 슬라이드 1280×720 한 화면 안에. 군더더기 금지.

## 퀴즈 정답·해설 (slide 14)
- Q1 "AI가 수업을 한 번에 다 만들어 준다" → **X**. 해설: 거시 설계도 후 슬라이드를 한 장씩, 교사가 흐름을 통제.
- Q2 "완성한 슬라이드들의 전체 흐름을 점검하고 설계도를 고치는 단계는?" → **flow-check**. (보기: blueprint / slide-compose / flow-check)
- Q3 "핵심 정리(정답)를 슬라이드 처음부터 보여준다" → **X**. 해설: 탐구 우선 — 활동 후 reveal로 펼침.

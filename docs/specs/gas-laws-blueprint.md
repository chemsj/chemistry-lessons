# 기체의 성질 ① 기체 법칙 (보일·샤를·아보가드로) — blueprint
- 폴더: `me-1-states/01-gas-laws/`  · 대상: 고2~3 진로선택(물질과 에너지) · 분량: 1차시(50분)
- 성취기준: **[12물에01-01]** 중 압력·온도·양과 부피의 관계 (2차시 중 1차시 / 2차시는 이상 기체 방정식·활용)
- 상태: planning · 최종수정: 2026-06-19

## 학습 목표
1. 보일 법칙으로 일정 온도에서 기체의 압력과 부피가 반비례함을 설명하고 계산할 수 있다.
2. 샤를 법칙으로 일정 압력에서 기체의 부피가 절대온도에 비례함을 설명하고, 절대온도(절대영도)의 의미를 안다.
3. 아보가드로 법칙으로 같은 온도·압력에서 기체의 부피가 양(mol)에 비례함을 설명할 수 있다.

## 대상 오개념
- 절대온도 0 K에서 부피가 0이 된다 → 실제 기체는 그 전에 액화(이상기체 가정의 외삽).
- 보일·샤를을 조건(등온/등압) 없이 "항상" 적용.
- 샤를 법칙 계산에 섭씨온도 사용(절대온도 K를 써야).
- 압력–부피를 비례로 착각(실제는 반비례).
- 기체 종류가 다르면 같은 조건에서도 부피가 다르다(아보가드로: 같은 조건 = 같은 부피).

## 서사 아크
도입(**같은 풍선**에 ①불기·②진공 감압을 **조별 직접 체험** + ③액체 질소를 **영상**으로 관찰 → 세 자극 모두 부피 변화 → "원인과 규칙은?" 종합. *로컬 체험 + 온라인 영상의 유기적 연결*) → 보일(압력↑→부피↓, 입자 시뮬·그래프) → 샤를(온도↑→부피↑, 절대온도 도입, 그래프로 절대영도 외삽) → 아보가드로(양↑→부피↑, 같은 조건 같은 부피) → 종합(세 법칙 한눈에 V∝1/P·V∝T·V∝n) → 형성평가(오개념 표적) → 마무리(마인드맵).

## 참고·출처 (검증값)
- 보일 `P₁V₁=P₂V₂` / 샤를 `V₁/T₁=V₂/T₂`, `T(K)=t(℃)+273`, 절대영도 −273 ℃ / 아보가드로 `V∝n`, 0 ℃·1기압에서 1 mol = 22.4 L.
- 시뮬 부피는 **상댓값**(입자 수·공간)으로 표기 명시.
- 미래엔 p8~12 (타 출판사 대응 페이지: `matter-energy-textbook-index.md`). 공식 권장 탐구(별책9): 이상 기체 방정식으로 기체 종류 확인.
- **도입 준비물**: (조별 실험 도구) 풍선 · 간이 진공 펌프 용기 — 학생이 ①불기·②감압을 직접 수행 / (영상) **액체 질소 풍선 시연 영상**(공개 영상 링크 또는 교사 시연 촬영 — 위험해 직접 X). 슬라이드는 **활동 안내 + 관찰 + 영상 + 종합 발문** 중심(현장 사진은 선택).
- 도입 설계: **로컬(조별 직접 체험) + 온라인(영상)** 을 한 화면에서 같은 '부피 변화'로 수렴시켜 본문 세 법칙으로 연결.

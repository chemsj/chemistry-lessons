# 르 샤틀리에 원리 — 평형 이동 (도입 차시) — blueprint
- 폴더: `demo-le-chatelier/`  · 대상: 화학Ⅱ(고2~3) · 분량: 1차시(50분)
- 성격: **독립 예시(기존 chem-3-equilibrium 02·03과 분리)** · 교사 연수 시연용
- 상태: planning · 최종수정: 2026-06-17
- 중심 앵커 반응: **N₂O₄(g) ⇌ 2NO₂(g)** (무색 ⇌ 갈색)

## 학습 목표
1. 평형 상태의 계에 자극(농도·압력·온도)을 주면 **그 자극을 줄이는(완화하는) 방향**으로 평형이 이동함(르 샤틀리에 원리)을 설명할 수 있다.
2. N₂O₄ ⇌ 2NO₂에서 농도·압력·온도 변화에 따른 **이동 방향을 예측**하고 **색 변화(무색↔갈색)**로 설명할 수 있다.
3. **온도 변화만 평형 상수 K를 바꾸고**, 농도·압력 변화는 K를 바꾸지 않음을 구분할 수 있다.

## 대상 오개념
- (부분 상쇄) 평형 이동이 자극을 **완전히** 상쇄한다 → 실제는 **일부만** 상쇄(반응물 더 넣으면 일부만 소모되고 여전히 더 많음).
- (압력) 압력/부피를 바꾸면 **항상** 이동한다 → 기체 **Δn=0이면 이동 없음**(이 반응은 Δn=+1이라 이동).
- (K 혼동) 농도·압력 변화도 K를 바꾼다 → **온도만** K를 바꾼다.
- (온도) 온도를 올리면 **항상 정반응** → 흡열/발열에 따라 다름(이 반응은 흡열 정반응 → 가열 시 정반응).
- (촉매) 촉매가 평형을 이동시킨다 → 촉매는 **도달 속도만** 빠르게, 평형 위치·K 불변.

## 서사 아크
도입(평형인 계를 흔들면 색이 어디로? 동기) → 탐구1 농도(반응물 추가 → 일부만 소모 관찰 → **원리 발견**) → 탐구2 압력/부피(기체 분자 수 적은 쪽) → 탐구3 온도(흡열 정반응·색 언어·**온도만 K 변경**) → 오개념 직면(부분 상쇄·촉매) → 형성평가 4문항 → 마무리 마인드맵.

## 참고·출처 (검증값 — 학생 화면 미언급 근거는 demo-le-chatelier/출처.md에)
- ΔH°(정반응, N₂O₄→2NO₂) = **+57.2 kJ/mol** (흡열). [2·ΔHf(NO₂ +33.2) − ΔHf(N₂O₄ +9.16)]
- 기체 화학량론: 좌 1몰(N₂O₄) ⇌ 우 2몰(NO₂), **Δn = +1**.
- **Kp(25℃) ≈ 0.15** (≈0.14, ΔG°≈+4.7 kJ/mol). 흡열이므로 **온도↑ → K↑**.
- 색: N₂O₄ 무색, NO₂ 적갈색. 가열→갈색 진함, 냉각→옅음. 압축(부피↓)→평형 도달 후 옅어짐(압축 직후 일시적 진해짐은 교사용 노트로).
- 시뮬 농도는 **상댓값(입자 개수 비례)** 으로 표기 명시. Q=[NO₂]²/[N₂O₄] 비로 방향·K 복귀를 일치시킴.
- 색 언어 고정: **흡열=빨강(정반응) · 발열=파랑(역반응)**.

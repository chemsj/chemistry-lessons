## 활용법
1. 단원 작업 시 **미래엔(주교과서) 해당 페이지를 Read로 심층 분석** + 위 표로 타 출판사 같은 주제를 찾아 **탐구 활동·오개념·설명 방식 비교**.
2. **비상(133MB)은 직접 Read 불가** → 해당 페이지만 분리 후 Read:
   ```bash
   pdfseparate -f <시작> -l <끝> "비상교과서.pdf" /tmp/b_%d.pdf
   pdfunite /tmp/b_*.pdf /tmp/bisang_part.pdf   # 합본
   # 그 뒤 /tmp/bisang_part.pdf 를 Read
   ```
3. 교사용 교과서·지도서는 같은 폴더(답·지도 팁·평가 자료).

- **천재**: 대단원 안에 중단원(1·2) → 소단원(01·02·03) 2단계.

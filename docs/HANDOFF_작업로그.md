## 3) 02 단원 핵심 사항 (이미 반영됨)
- 6슬라이드: 도입 / CoCl₂ 온도 실험(미시 시뮬) / 온도-K 데이터 탐구 / 르샤틀리에 종합표 / 형성평가 / 마무리.
- CoCl₂ 시뮬: **실측 K(25℃)=1.7×10⁻³**(U.Colorado/Purdue 데모) 앵커 + van 't Hoff(흡열 ΔH≈+50kJ/mol). 표시는 **몰 농도(mol/L)**이며 그 값을 식에 넣으면 그대로 Q가 나옴(Q=K at 평형). 평형 f는 Q(f)=K(T) 이분법 해. 농도 표기는 **대괄호 `[ ]`(교과서식)** — `c( )`는 미학습이라 안 씀. 그래프=코발트 농도(mol/L), Cl⁻은 다량(정성).
- 온도-K 데이터(슬라이드3): **MBL 측정처럼 한 점씩** 누적(자동시작X, "측정 시작"→"다시 측정"). 흡열 N₂O₄(동아 0.15→1700)/발열 NH₃(Wikipedia 4.34e-3→1.45e-5) 실측 양끝 + van't Hoff 보간 6점, **K는 10의 지수 표기**.
- 르샤틀리에 종합(슬라이드4): **인터랙티브 예측기**(조건 버튼→마커·화살표·결과) + 우측 표는 **탐구하면 채워짐**(`.lkcell` 잠금). 예시 `2A(g)⇌B(g)`. 조건="온도 증가/감소", 이동방향·K는 정반응 기준 흡열/발열 분기.
- 핵심 정리·마무리: 모두 **칩 흐름**(아이콘→화살표→색칩, 흡열/냉각=빨강·발열/가열=파랑) 시각화. 마무리=2단(좌 오늘 배운 내용/우 다음 차시).
- 그래프 라벨 **흰 칩**(선·숫자 겹침 방지). 펄스 애니메이션은 inline+고정높이로 스크롤바 깜빡임 방지. 2단 카드 **높이 정렬**(flex:1).
- ※ 암모니아 슬라이드는 **02에서 빼고 03으로 이동**함.
- ※ 위 패턴들은 모두 **README §24~30** + 템플릿(`.flowrow`/`.lkcell`/`lcpulse`) + 프로토콜 체크리스트에 반영됨 → 다음 단원에서 재사용.

## 4) 03 단원 핵심 사항 (이미 반영됨, 데이터는 03/출처.md)
- 8슬라이드. 각 사례 **개별 슬라이드 + 실제 데이터 시뮬**:

  - 해양 산성화: `pH=8.2−0.57·log10(CO₂/280)` (420ppm→8.10, [H⁺]+26%). NOAA.
  - 치아: 임계 pH 5.5(법랑질)/4.5(불소) 기준 탈회·재광화.

## 4-2) Ⅳ-1 「화학 전지」 핵심 사항 (이미 반영됨, 출처는 해당 폴더 출처.md)
- 교과서 근거: `교과서파일/화학2/..._4_전기화학과이용.pdf` **180–187쪽**.
- 8슬라이드: 도입 / ①이온화 경향 / ②볼타 전지 / ③다니엘 전지 / ④실용 전지 / ⑤종합 비교 / ⑥형성평가(4문항) / 마무리.
- 신규 캔버스 시뮬(템플릿 헬퍼 재사용 + 전용 함수):
  - **이온화 경향**(`initIonize`/`ionCase`): 4조합 버튼 → 반응 O/X 판정 + 서열막대 강조(.ox/.red). 반응 = 판 금속이 이온 금속보다 이온화경향 클 때.
  - **볼타**(`initVolta`): 도선 전자(Zn→Cu)·Zn²⁺ 용출·H⁺→H₂ 기포. **분극 토글** → 기포 점착·전압강하 그래프(상댓값 1.10→0.32).
  - **다니엘**(`initDaniell`): 전자·**염다리 이온**(양이온→CuSO₄/음이온→ZnSO₄)·질량칩(Cu/Zn=0.97=63.5/65.4)·CuSO₄ 색 옅어짐·Cu 석출.
  - **실용 전지**(`initBatt`/`battMode`): 방전/충전 전자 방향 반전(1차전지 충전 불가 설명).
- 전용 CSS(`</style>` 직전): `.ionseries/.msym/.verdict/.massrow/.masschip/.eleceq/.celltag`. 전용 JS는 `/* 시작 */` 직전 블록.
- 검증: node --check 통과, 콘솔 0, 활동 슬라이드(1–6) 넘침 0, 도입(76)·마무리(53)는 single 소량 스크롤(허용). 채점 4/4 정상.

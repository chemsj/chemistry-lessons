## 3) 02 단원 핵심 사항 (이미 반영됨)
- 6슬라이드: 도입 / CoCl₂ 온도 실험(미시 시뮬) / 온도-K 데이터 탐구 / 르샤틀리에 종합표 / 형성평가 / 마무리.
- CoCl₂ 시뮬: **실측 K(25℃)=1.7×10⁻³**(U.Colorado/Purdue 데모) 앵커 + van 't Hoff(흡열 ΔH≈+50kJ/mol). 표시는 **몰 농도(mol/L)**이며 그 값을 식에 넣으면 그대로 Q가 나옴(Q=K at 평형). 평형 f는 Q(f)=K(T) 이분법 해. 농도 표기는 **대괄호 `[ ]`(교과서식)** — `c( )`는 미학습이라 안 씀. 그래프=코발트 농도(mol/L), Cl⁻은 다량(정성).
- 온도-K 데이터(슬라이드3): **MBL 측정처럼 한 점씩** 누적(자동시작X, "측정 시작"→"다시 측정"). 흡열 N₂O₄(동아 0.15→1700)/발열 NH₃(Wikipedia 4.34e-3→1.45e-5) 실측 양끝 + van't Hoff 보간 6점, **K는 10의 지수 표기**.
- 르샤틀리에 종합(슬라이드4): **인터랙티브 예측기**(조건 버튼→마커·화살표·결과) + 우측 표는 **탐구하면 채워짐**(`.lkcell` 잠금). 예시 `2A(g)⇌B(g)`. 조건="온도 증가/감소", 이동방향·K는 정반응 기준 흡열/발열 분기.
- 핵심 정리·마무리: 모두 **칩 흐름**(아이콘→화살표→색칩, 흡열/냉각=빨강·발열/가열=파랑) 시각화. 마무리=2단(좌 오늘 배운 내용/우 다음 차시).
- 그래프 라벨 **흰 칩**(선·숫자 겹침 방지). 펄스 애니메이션은 inline+고정높이로 스크롤바 깜빡임 방지. 2단 카드 **높이 정렬**(flex:1).
- ※ 암모니아 슬라이드는 **02에서 빼고 03으로 이동**함.
- ※ 위 패턴들은 모두 **README §24~30** + 템플릿(`.flowrow`/`.lkcell`/`lcpulse`) + 프로토콜 체크리스트에 반영됨 → 다음 단원에서 재사용.

## 4) 04 단원 「일상 속 평형 이동」 핵심 사항 (2026-06-09 개편 반영, 데이터는 04/출처.md)
- 8슬라이드. **모든 사례를 단일 슬라이드 "사례 학습" 포맷으로 통일**(좌 시뮬 + 우 그래프 + 공유 컨트롤 + 2박스 헤더 + 핵심정리 칩 + 종합) → **README §31** 골격.
  - 암모니아(하버법): 공정 도식(압축·반응·냉각·재순환) 시뮬 + 수득률 곡선, `.meters` 3열(평형 수득률·속도·실효 생산), 실제 산업조건(~450℃/200~300atm) 타협 verdict.
  - 헤모글로빈: 폐↔조직 순환 시뮬(정맥6→동맥8→조직6 O₂) + Hill식(P50 26.6, n 2.7) 해리곡선.
  - 탄산음료: 병 CO₂ 용해 시뮬(개봉=압력↓ 단방향, 용존 CO₂ 단조감소, 평형 시 기포 0) + 용해도 곡선. **헨리 법칙 명칭 미사용**("용해는 발열·압력 비례"로 설명, §34).
  - 해양 산성화: 바다 단면 시뮬 + `pH=8.2−0.57·log10(CO₂/280)`(420→8.10, 1000→7.88, [H⁺]+107%≈2배=−log₁₀2). **H⁺+CO₃²⁻→HCO₃⁻** 연결고리(CO₃²⁻ 입자 감소·캡션·3칩)로 골격 형성 곤란까지 연결. NOAA.
  - 치아: 법랑질 단면 시뮬 + 탈회 곡선(임계 pH 5.5/불소 4.5). 불소=OH⁻→F⁻ 치환→플루오린화인회석. **타액 이온 막대그래프(상대·모식)**: H⁺·OH⁻·Ca²⁺·PO₄³⁻ pH 연동 — **자유이온 vs 총용해량 구분**(Ca²⁺↑ 방출/PO₄³⁻·OH⁻↓ H⁺가 소모), §32. OH⁻도 함께 표기(pH7서 H⁺=OH⁻).
- 형성평가: **4문항 2×2 그리드(single) + 상단 점수판**(산업·생명·환경·건강 고르게), §33. 마무리: 2단(오늘 배운 5사례 / 공통 원리 칩+종합).
- **공개**: 대단원 허브 카드 활성화(준비 중→공개) + `.gitignore`에서 04 제외 해제 + commit/push(`8a5c5b4`). 라이브 200 확인 → `https://chemsj.github.io/chemistry-lessons/chem-3-equilibrium/04-equilibrium-in-life/`. (탭 `<title>`은 "03. …" 유지 — 사용자 확인필.)

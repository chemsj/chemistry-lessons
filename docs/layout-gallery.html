슬라이드 레이아웃 갤러리

수업 슬라이드를 만들 때 상황에 맞는 레이아웃을 먼저 고르세요. 실물 견본은 docs/lesson_template.html에 같은 이름으로 들어 있습니다.

＋ 새 레이아웃·예시 추가하는 법

  1. docs/lesson_template.html에 <section class="slide …"> 한 장을 추가합니다 (CSS 클래스는 아래 각 카드의 코드 참고).
  2. 이 페이지의 레이아웃 카드(.lay) 블록 하나를 복사해 와이어프레임과 설명을 채웁니다.

또는 Claude Code에 “이 레이아웃을 이런 예시로 갤러리에 추가해줘”라고 요청하면 등재됩니다. 새 레이아웃이면 CSS(lesson_template.html + deck-shell.html)까지 함께 추가합니다.

1단 .single

적합 — 전체 폭을 쓰는 단일 메시지

예시 — 도입 동기, 마무리 마인드맵

2단 (기본)

적합 — 발문 + 탐구(개념 학습, 가장 흔함)

예시 — 좌 질문/정리 + 우 입자 시뮬

3단 .cols3

적합 — 셋을 나란히 비교 · 3단계 과정 · 세 사례

예시 — 흡열 / 발열 / 촉매, 반응물 · 정반응 · 생성물

상하 분할 .split

적합 — 위 제시 → 아래로 가로로 넓게 전개

예시 — 위 큰 그래프 + 아래 데이터 표

헤더+2단 .headcols

적합 — 공통 맥락을 위에 고정 + 좌우 활동

예시 — 상단 반응식(N2O4 ⇌ 2NO2) + 좌 시뮬 / 우 그래프

2단 역비율 .cols2wide

적합 — 시뮬·그림이 주인공, 설명은 보조

예시 — 좌 큰 입자 시뮬 + 우 짧은 관찰 포인트

흡열 = 빨강 · 발열 = 파랑 색 언어와 정답/오답 색은 모든 레이아웃 공통입니다 · 토큰은 lesson_template.html 기준

1 온도 변화

온도를 바꾸면 — 염화 코발트(II)

생각 열기

[Co(H2O)6]2+(붉은색) + 4Cl− ⇌ [CoCl4]2−(파란색) + 6H2O  (정반응 = 흡열) 온도를 올리고 내리면 색은 어떻게 변할까?

슬라이더로 가열/냉각하며 색·입자·K 변화를 관찰하세요.

2 데이터 기반 탐구

데이터로 확인 — 온도와 평형 상수

생각 열기

실제 측정한 온도별 평형 상수 K 자료에서 규칙을 찾아보자. 흡열 반응과 발열 반응은 온도에 따라 K가 어떻게 달라질까?

흡열/발열 버튼을 누르면 온도를 올리며 K를 한 점씩 측정하듯 표·그래프에 데이터가 쌓입니다. 두 경우의 경향을 비교해 보세요.

핵심 정리
🔺 흡열 반응 온도↑ →정반응 우세 →K 증가 ↑
🔻 발열 반응 온도↑ →역반응 우세 →K 감소 ↓
농도·압력은 K 불변 — 온도만이 K를 바꾼다
실제 데이터 · 표 + 그래프
🌡 측정 대기
log K – 온도

## 교과서 근거
- 본문·반응식·그림(Ⅳ-1~Ⅳ-8)·탐구 활동은 폴더 내
  `교과서파일/화학2/화학2_교과서PDF_4_전기화학과이용.pdf` **180–187쪽**을 분석해 재구성.
- 핵심 반응식(교과서 그대로):
  - 볼타: (−) `Zn → Zn²⁺ + 2e⁻` / (+) `2H⁺ + 2e⁻ → H₂` / 전체 `Zn + 2H⁺ → Zn²⁺ + H₂`
  - 다니엘: (−) `Zn → Zn²⁺ + 2e⁻` / (+) `Cu²⁺ + 2e⁻ → Cu` / 전체 `Zn + Cu²⁺ → Zn²⁺ + Cu`
  - 이온화 경향: K>Ca>Na>Mg>Al>Zn>Fe>Ni>Sn>Pb>(H)>Cu>Hg>Ag>Pt>Au (그림 Ⅳ-1)

## 시뮬레이션 수치 — 상댓값/근거
| 시뮬 | 값 | 성격 |
|---|---|---|
| 볼타 전압계 | 1.10 → 최저 0.32 (분극 시) | **상댓값**. 분극에 의한 전압 강하 ‘경향’을 표현(절대값 아님) |
| 다니엘 전압계 | 1.10 V 고정 표시 | 다니엘 전지 표준 기전력(약 1.1 V) 참고값 |
| 다니엘 질량 변화 | 구리/아연 = 0.97 | 원자량 비 **63.5/65.4 ≈ 0.97** 반영(아연판 감소 : 구리판 증가) |
| 이온 용액 색 | CuSO₄ 파랑, ZnSO₄·AgNO₃ 무색 | 정성적 표현(반응 진행 시 파란색 옅어짐/짙어짐) |

- 전압·기포 수·입자 개수는 **교육용 정성 모형**으로, 실제 측정 단위가 아님(화면에 별도 표기 불필요한 상댓값).
